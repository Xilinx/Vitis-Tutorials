Vitis™ Platform Creation Tutorials

Introduction¶

Xilinx provides base platforms for Xilinx Evaluation Boards to allow for quick out-of-the-box experimentation.

For additional customization or more in-depth development on Xilinx boards, or to develop on custom board, you might want to create a fully customized hardware platform.

These tutorials introduce the basics of platform creation and provide step-by-step tutorials for creating custom Vitis and Vitis AI™ ready platforms.

Tutorial Description
Vitis Platform Overview This is an overview of the Vitis platform creation concepts and steps.
Vitis Custom Embedded Platform Creation Tutorial on MPSoC This tutorial demonstrates how you can create a custom Vitis platform on ZCU104 demo board and validate the platform.
Versal Custom Platform Creation Tutorial This tutorial demonstrates how you can create a custom Vitis platform on VCK190 demo board and validate the platform.

Design Tutorials¶

Tutorial Description
Custom Embedded Platform Creation Example on KV260 Extend the ZCU104 custom embedded platform creation example to the popular MPSoC board: Kria KV260.

Feature Tutorials¶

These tutorials illustrate various platform features and how you can incorporate them into your own custom platforms.

Incorporating Stream Interfaces This tutorial demonstrates how you can stream data between Vitis kernels and AXI stream connections exposed in the platform.
