##############################################################################
Vitis Platform Creation Tutorials
##############################################################################





Xilinx provides base platforms for Xilinx Evaluation Boards to allow for quick out-of-the-box experimentation. For additional customization or more in-depth development on Xilinx boards, or to develop on custom board, you may wish to create a fully customized hardware platform.

These tutorials introduce the basics of platform creation and provide step-by-step tutorials for creating custom Vitis and Vitis AI ready platforms.



.. sidebar:: More Information

    See Vitis Development Environment on `xilinx.com <https://www.xilinx.com/products/design-tools/vitis.html>`_.
