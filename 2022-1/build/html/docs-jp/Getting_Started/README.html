Vitis™ 入門チュートリアル

xilinx.com の Vitis™ 開発環境を参照
 xilinx.com の Vitis-AI™ 開発環境を参照
このセクションでは、Vitis の入門チュートリアルを紹介します。

  • Vitis の概要および入門 :

    • カーネル開発、ホスト ソフトウェアの作成、エミュレーション、インプリメンテーション、解析などの Vitis ワークフローの概要。

      • Vitis がソフトウェア、アクセラレーション、ML 開発を単一の開発プラットフォームで統合する方法について説明します。

  • Vitis HLS :

    • Vitis HLS (高位合成) 環境内から個々のハードウェア アクセラレータを最適化、インプリメンテーション、およびユニット テストする方法について詳しく説明します。

  • Vitis ライブラリ:

    • L1 Vitis ライブラリ エレメントを使用して独自のデザインを構築する方法を学びます。説明のため、FFT の例が選択されていますが、同様のフローに従って、ほかのライブラリ関数を使用することもできます。

  • Vitis プラットフォーム

    • このチュートリアルでは、Vivado のカスタマイズ可能なサンプル デザインからの Versal Extensible Platform とプリビルド済みの Linux 共有イメージを使用して、Versal に Vitis エンベデッド アクセラレーション プラットフォームをすばやく作成して検証する方法を示します。
